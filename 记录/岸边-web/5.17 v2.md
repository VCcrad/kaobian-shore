## 实现说明

1. 专业大类映射表（12 个大类）  
覆盖：计算机、电子信息、电气、机械、土木建筑、经管、法学、教育、医学、中文传媒、思政、心理等。  
例如输入 「计算机」 会自动扩展匹配：软件、网络工程、大数据、信息安全、人工智能、信息技术 等。

2. 双向模糊包含

- 输入「电子」→ 能命中「电子信息工程」
- 岗位要求「计算机」→ 输入「计算机科学与技术」也能命中
- 在标题、专业字段、标签全文检索

3. 多关键词（空格分隔，OR 逻辑）

- `计算机 电子`：命中计算机类 或 电子信息类任一即可
- 输入即筛，无需点搜索按钮

4. 空结果提示  
有专业关键词但无结果时显示：  
「没有找到相关专业的岗位，换个关键词试试吧～」

> 筛选在前端完成（数据已在首页加载），无需改后端 API；岗位多时可再做成服务端搜索。

---

## 怎么测

1. 刷新 [http://localhost:3000](http://localhost:3000/)
2. 在搜索框输入 计算机 → 应看到含「计算机」「软件」等专业的岗位
3. 输入 电气 → 命中电气、电力、机电等相关岗位
4. 输入 计算机 电子 → 两类岗位都会出现
5. 输入 考古（若无相关岗位）→ 显示友好空状态文案

后续要加新大类，在 `lib/major-search.js` 的 `MAJOR_CATEGORIES` 里追加 `roots` 和 `aliases` 即可。
