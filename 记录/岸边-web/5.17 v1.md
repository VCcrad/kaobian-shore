## 新增内容

1. 删除接口 `app/api/jobs/[id]/route.js`

- `DELETE /api/jobs/3` → `prisma.job.delete({ where: { id: 3 } })`

2. 列表接口 `app/api/jobs/route.js`

- 增加 `GET /api/jobs`，供后台拉取已发布岗位

3. 管理后台 `app/admin/page.js`

- 三栏下方增加 「已发布岗位列表」
- 每条右侧有红色 「下架」 按钮
- 下架成功后右侧操作台显示绿色 「下架成功！」，列表自动刷新
- 发布成功也会刷新该列表

## 测试步骤

1. 打开 [http://localhost:3000/admin（密码](http://localhost:3000/admin%EF%BC%88%E5%AF%86%E7%A0%81) `123456`）
2. 向下滚动，在 「已发布岗位列表」 里找到要删的岗位
3. 点红色 「下架」 → 应提示「下架成功！」，该行立刻消失
4. 打开 [http://localhost:3000](http://localhost:3000/) 刷新首页 → 该岗位也应不再显示

若 `npm run dev` 没在跑，先在项目根目录执行一次再测。
