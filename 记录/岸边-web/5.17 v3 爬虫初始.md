已在项目根目录创建 `scripts/spider-safe.js`，并在 `package.json` 里加了快捷命令。

## 脚本包含的防封能力

|规范|实现|
|---|---|
|UA 随机池|8 条 Chrome / Edge / Safari / Firefox（Win / Mac / iOS / Android）|
|随机延迟|`sleep(ms)` + 每次请求前后各等待 3–7 秒（随机）|
|禁止并发|仅单次串行 `axios.get`|
|请求头|`Accept`、`Accept-Language: zh-CN,zh;q=0.9`、`Referer`（学校首页）等|
|熔断|403 / 429 / 连续失败 2 次 → 立即停止后续请求|
|解析|Cheerio 提取标题、链接、发布时间（含通用选择器兜底）|

使用前请在脚本顶部修改：

- `SCHOOL_HOME_URL` → 目标学校官网首页
- `TARGET_LIST_URL` → 真实招聘列表页地址

---

## 终端怎么运行

在项目根目录执行：

cd "d:\工作相关\anBian-web"

node scripts/spider-safe.js

或使用 npm 脚本：

npm run spider:safe

终端会看到：随机 UA、延迟倒计时、解析结果预览；若遇 403 会打印熔断提示并安全退出。

> 提醒：请仅对有权抓取或已获授权的站点使用；模板 URL 为占位，正式使用前务必换成真实列表页并调整 Cheerio 选择器。
